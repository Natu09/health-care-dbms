# AHS Database README

## Make sure you have installed node.js
https://nodejs.org/en/

## First cd into the "frontend" folder

### To install dependencies

`npm i`

### To run the application locally

`npm start`

### Access the login page
Make sure that the url displays <br /> 
`localhost:3000/login`
